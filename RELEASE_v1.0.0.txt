Installation & Setup
====================

Prerequisites
- Python 3.13+
- PostgreSQL 12+
- pip/virtualenv

Installation Steps:
1. git clone https://github.com/Muhammad-Bilal-77/visiontrack-backend.git
2. cd visiontrack-backend
3. python -m venv venv
4. venv\Scripts\activate (Windows) or source venv/bin/activate (Linux/Mac)
5. pip install -r requirements.txt
6. cp .env.example .env (configure your credentials)
7. python manage.py migrate
8. python manage.py createsuperuser
9. python manage.py runserver

Key Configurations
- Django Settings: visiontrack/settings.py
- Database: PostgreSQL (configurable via .env)
- Media Root: media/ directory
- Static Files: static/ directory (configured)

API Highlights
==============

Authentication
- POST /auth/jwt/create/ → Get JWT token
- POST /auth/jwt/refresh/ → Refresh access token
- POST /auth/jwt/blacklist/ → Logout

Employee Management
- GET /api/employees/ → List all employees
- POST /api/employees/ → Create new employee
- GET /api/employees/<id>/ → Employee detail
- PUT /api/employees/<id>/ → Update employee
- DELETE /api/employees/<id>/ → Delete employee

Attendance Operations
- POST /api/attendance/check-in/ → Employee check-in
- POST /api/attendance/check-out/ → Employee check-out
- GET /api/attendance/today/ → Today's records
- GET /api/attendance/history/<id>/ → Employee history
- GET /api/attendance/by-date/?date=YYYY-MM-DD → Date-wise records

User Management (Djoser)
- POST /auth/users/ → Create user
- GET /auth/users/me/ → Current user
- POST /auth/users/set_password/ → Change password
- POST /auth/users/reset_password/ → Reset password
